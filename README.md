## 🎯 Learning Objectives

By working through this practice test, you will review and reinforce your understanding of:

- Python looping constructs (`for`, `while`, `range`)
- Nested loops and iteration patterns
- Functions: definitions, arguments, return values, and default parameters
- Python data structures: lists, tuples, dictionaries, and sets
- List and dictionary operations (slicing, comprehensions, iteration)
- String methods and f-strings
- Lambda functions and operator precedence
- Object-oriented programming basics (`class`, `__init__`, instance attributes)
- Algorithmic approaches: exhaustive enumeration and approximation

## 📋 Instructions

1. Open `writing/practice_test.md`
2. Work through each question carefully — read all four options before choosing. While you could easily copy and paste the given code into Python to read the output, this will not be helpful preparation for a paper-based midterm exam. Instead, try to reason your way through the problem *without* copying and pasting.
3. Record your chosen answer next to each question by replacing the `TODO` marker
4. After completing the practice test, check your answers by copying and pasting the code to determine correctness.
5. Complete the reflection in `writing/reflection.md`

>**Tip:** For code-based questions, trace through the code by hand before selecting an answer. Write out variable values at each step. Play the part of Python yourself!

The grade that a student receives on this assignment will have the following components.

* **Mastery of Technical Knowledge and Skills [up to 90%]**: Students will receive a portion of their assignment grade when their program implementation reveals that they have mastered all of the technical knowledge and skills developed during the completion of this assignment. As a part of this grade, the instructor will assess aspects of the programming including, but not limited to, the completeness and the correctness of the program and the use of effective source code comments.
